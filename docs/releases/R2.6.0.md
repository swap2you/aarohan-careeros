## Summary

Controlled application assistance for Greenhouse, Lever, and Ashby hosted pages. Aarohan prepares field mappings, rechecks duplicate and vendor risk, and stops before external submit. Manual mode unchanged. Autonomous remains HTTP 403.

- Prohibited sites: LinkedIn, Indeed, Glassdoor, Dice → Manual only
- Unsupported ATS (Workday, etc.) → Manual fallback
- Job detail ATS/mode panel
