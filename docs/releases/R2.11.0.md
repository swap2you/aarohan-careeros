## Principles

- Local-first remains default; cloud is opt-in future phase
- Secrets in managed vault (not Git, not Drive)
- HttpOnly session cookies, CSRF for mutating browser routes
- Object storage for generated documents; workers for schedules
- No multi-tenant code in local app until ADR-001 execution
