# R2.9 — Ask Aarohan and OpenAI TTS

**Tag:** `r2.9.0`

## Summary

Read-only Ask Aarohan over jobs, applications, companies, interviews, and Gmail signals with citations. OpenAI TTS for answers with cache, cost recording, and graceful fallback when no API key.

## Deliverables

- `POST /api/ask` — rule-based Q&A, secret-query blocking, uncertainty statements
- `POST /api/tts` + `GET /api/tts/file/{digest}` — cached MP3 under `generated_root/tts/`
- Ask Aarohan page with read-aloud (authenticated blob fetch)
- Config: `ask_aarohan_sql_mode`, `openai_tts_model` (SQL mutations blocked by default)

## Security

- No OpenAI key in browser bundles
- Token/secret questions refused
- TTS files require session cookie
