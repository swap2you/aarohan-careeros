# Interview Grilling Machine

The system must create a job-specific preparation workspace.

## Sections
- Company and industry briefing
- Role outcome map
- Resume-to-JD gap analysis
- Recruiter-screen questions
- Hiring-manager questions
- Technical questions
- API, automation, performance, CI/CD questions
- System-design and platform questions
- Quality strategy and governance questions
- AI-enabled QE questions
- Agentic AI, prompting, context engineering, MCP, LLM evaluation, RAG evaluation, guardrails, observability, and human-in-the-loop questions when relevant
- Leadership and stakeholder questions
- Behavioral questions
- Verified STAR story mapping
- Honest gap answers
- Exercises
- Take-home simulations
- Answer scoring
- Weak-area tracker
- Seven-day preparation plan
- Voice mock-interview prompt

## Truthfulness
When the candidate lacks direct experience:
- say so;
- bridge from adjacent experience;
- provide a small portfolio exercise;
- never fabricate production experience.
