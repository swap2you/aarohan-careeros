# Local Environment Template

Cursor must create `.env.local` automatically if missing, but it must never place the Google client secret directly in that file.

```dotenv
APP_ENV=local

CAREER_GMAIL_ADDRESS=swapnilpatil.tech@gmail.com
GOOGLE_CLOUD_PROJECT_ID=aarohan-careeros-500722
GOOGLE_CLOUD_PROJECT_NUMBER=558756512850
GOOGLE_DRIVE_ROOT_FOLDER_ID=1EaueVpEFOkZE-_9EKrY-_xdcJgY1Jkqr
GOOGLE_OAUTH_CLIENT_JSON_PATH=C:\AarohanSecrets\google-oauth-client.json

ENABLE_SCHEDULED_WORKFLOWS=false
ENABLE_FINAL_APPLICATION_SUBMISSION=false
ENABLE_EXTERNAL_EMAIL_SEND=false

TEST_EMAIL_ALLOWLIST=swap2you@gmail.com,patilsaci@gmail.com,sriswapnilpatil@gmail.com,patilswapnilqa@gmail.com
```

Other local secrets must come from the current local secret mechanism already implemented by the repository or be generated by the local initialization script.
