# Cowork UAT

Test these user journeys:
1. ingest a fresh public job;
2. normalize and deduplicate;
3. verify salary and remote scoring;
4. generate a grounded packet;
5. approve, edit, hold, and reject;
6. mark submitted;
7. ingest recruiter response;
8. create full interview pack;
9. run exercises and track weak areas;
10. ingest a consulting lead;
11. generate service recommendation and proposal;
12. inspect analytics and audit logs;
13. test failures and retries;
14. confirm no external action occurs without approval.

Return:
- PASS or FAIL;
- failed scenarios;
- evidence;
- usability defects;
- missing requirements;
- release recommendation.
