# Codex Audit

Audit the repository and running environment independently.

Validate:
- design conformance;
- secrets;
- prohibited automation;
- source adapters;
- scoring;
- salary and remote logic;
- evidence grounding;
- resume and document generation;
- approval boundaries;
- Interview Grilling Machine;
- consulting pipeline;
- authentication;
- AI budget caps;
- audit logs;
- migrations;
- Docker;
- CI/CD;
- backup and rollback;
- tests.

Do not trust Cursor's report.

Output:
- PASS or STOP;
- blockers;
- defects by severity;
- exact evidence;
- commands executed;
- actual results;
- exact repair instructions.
