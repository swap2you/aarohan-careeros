# Release Signoff Checklist

Decision:
- GO
- CONDITIONAL GO
- STOP

Required evidence:
- Codex PASS;
- Cowork UAT PASS or approved conditional PASS;
- no critical defects;
- no secrets;
- no prohibited automation;
- tests passing;
- backups tested;
- rollback tested;
- hard AI cap active;
- dashboard authenticated;
- claims grounded;
- approval boundaries enforced.
