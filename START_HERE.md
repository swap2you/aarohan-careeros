# Aarohan CareerOS — V3 Build Kit

## Purpose
Build a cloud-hosted, supervised Career Operations System that helps the candidate:
1. secure a stable technical leadership role with a target base salary of $200,000+ plus benefits;
2. prioritize fully remote U.S. roles;
3. generate evidence-grounded application packets;
4. prepare deeply for interviews;
5. build a parallel Quality Engineering / AI transformation consulting pipeline.

## Current status
This kit contains the complete product design, candidate profile model, repository instructions, Cursor build prompt, validation prompts, security rules, release gates, and one-week execution plan.

It is **not** a completed production application yet.

## One action to start
1. Create a private GitHub repository named `aarohan-careeros`.
2. Extract this ZIP into the repository root.
3. Open the repository in Cursor.
4. Send Cursor this exact instruction:

```text
Read START_HERE.md, AGENTS.md, all files under docs/, config/, career_vault/, prompts/, and validation/. Then execute prompts/CURSOR_MASTER_BUILD_PROMPT.md. Do not commit or deploy until the required validation gates pass.
```

## Reality check
A focused week can produce a deployable end-to-end MVP with automated tests and supervised workflows.

A genuinely hardened production system normally requires additional calibration, security testing, source reliability work, and operational monitoring after the first deployment. Do not confuse “deployed” with “fully proven.”
