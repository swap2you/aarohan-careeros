# Career Vault

This folder is the source of truth for resumes, interview preparation, and job scoring.

Rules:
- Only externally defensible facts may be used in resumes.
- Early-career dates and company names marked `TODO_VERIFY` must be confirmed before use.
- Do not include confidential employer code, tickets, URLs, client data, credentials, or internal documents.
- Do not include dual-employment details in public-facing material.
- Do not include immigration documents. Store only a manually verified work-authorization summary if needed.
